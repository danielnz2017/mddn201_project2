MDDN201 Project 2

Brief

Based on the proposal from project 1, I have decided to create a website prototype for internet topics. The website will briefly talk about what is internet, how does it work and online privacy. User will be able to discuss the relevant topics anonymously at the end. Purpose: The purpose of this website is to explain the internet in general and its risks in a simple and easy way because currently there are many online articles that talk about internet and online privacy but most of those articles are long and very technical. Internet is not built just for IT experts; internet is for everyone to use. Therefore, I would like to build a website that can be a potential solution for it. User Goals: The website aims to inform users about the how does the internet work and the common risks of online personal details being collected, traced or exposed. Also, the website also aims to provide a platform for internet users to discuss their thoughts and opinions. Audience: Anyone who wants to know or just interested in some basic stuff of the internet and maybe like to share their opinions and thoughts as well.

Day 1 on Project

Based on the personas and the user flow in Project 1, I quickly wrote down some basic structure of the website. And then I drew two rough concept wireframes for the home page. For the first concept, I want to have a navigation bar on the top, a ‘network’ looking background with an animated earth in the centre of the website with several ‘category’ button around it. For the second concept, I decided to keep the basic layout form concept 1 and simply the layout to just 4 round buttons with all the small buttons around them.

Day 2 on Project

After selected a concept (concept 2), I started to use HTML and CSS to make my first home page. However, I was not able to achieve what I wanted because my HTML and CSS skills are not mature enough to make something too complicated.

Day 3 on Project

Since I found out that we are allowed to use prototype tools, I chose to use Adobe XD. So, I decided to improve my original plans for the website layout. I quickly made some rough wireframes for just giving myself an idea of my next step. I decided to have a big button on each page and all the text and image will be randomly placed around the page. Users will be able to proceed to the next page by clicking the round button on each page. Each page will give user a topic related to the internet. It feels more like an “internet journey”. After that, I made some digital box frames and wireframes for my ideas. I have also tried out a bunch of logo design styles.

Day 4 on Project

Today, I started my actual prototype making process. I selected a free background image online for my home page. In the first page, I made a round button to try to imitate the “dial-up internet” style. The purpose of doing this is to make the user experience feels more like a “interesting journey” rather than “painful readings”. I made those colour bars around the button and animated them in order to make it stand out. In the next page called ‘Hello Internet’, I first place a button that links to the first page as a “disconnect” button for exit purpose. And then I used a bunch of social media icons and rainbows to represent the ‘fun’ part of the internet. However, instead of use round button as I planned, I used a rectangle button. User can proceed to next page by clicking it and users will need to use exactly the same way in each page. I also place a little cursor at the corner of the button to stimulate users to click it.

Day 5 on Project

Today, I suddenly feel the original website layout was missing something and it does not feel convenient neither. Therefore, I created a navigation bar on the top of the screen and then I gave each button an icon to represent its topic. I have also removed buttons in the middle of each page and replaced them with each page title. I decided to use 4 relevant icons for each page for representing the topic for the current page. I made diagrams for explaining how internet works. I also made a page for users to discuss topics anonymously and a reference/credit page for all the free image I used for decorating my website prototype. The background colour has been changed to dark blue as well.>

Day 6 on Project

I made diagrams for briefly illustrating how internet works last time but I feel it is probably not enough. So, I created animations for each diagram I created in order to facilitate the understanding of how internet works. I made 3 animated diagrams for ‘internet’ page and users can click button called ‘view’ to view the animation. Also, I added the clicking feedback to the navigation bar in order to indicate the current page.

Day 7 on Project

Today, I have changed an option called “cyberattack” to “data monitoring” on the navigation bar to make it more accurate. I have also finished the content on the ‘data monitoring’ page. In this page, I mainly choose to talk website cookies in the relation to web tracking and data collection. I also made a brief animation for how cookie works on the website and how web host and social media companies use these data. I have also changed the black arrow to a blue arrow as it was not very intuitive.

Day 8 on Project

Today, I have just updated my home page by changed the logo and rearranged the layout a little bit. There were two rectangles for each part but now I have deleted the rectangle behind the logo and extended another rectangle to the logo. I have also adjusted the proportion by making the logo bigger. And I placed the reference button at the corner of the page.

Day 9 on Project

Today, I have changed the way I present my background. My original plan was to use 4 relevant icons in each page to represent the current topic. However, I have decided to intergrade only one icon into the background to make it looks more natural.

Day 10 on Project

Today, I have decided to change a button called “data monitoring” on the navigation bar to “privacy issues” as I feel the late one could not reflect the topic I am trying to talking about in the page. Another, instead of choosing one icon and integrate it into the background, I have decided to use multiple icons because it looks a little bit empty when the page is scrolled down.

Day 11 on Project

Today, I have increased the transparency of each big ‘background icon’ in order to make them blend into the background. I have also added new content to the ‘privacy issues’ pages and relocated the diagram to the bottom of the page. In addition, I have finished the content in the ‘Data Security’ page. In this page, it talks about several ways of improving online privacy awareness and how to reduce the risk of data breaches. The anonymous profiled icon has also been updated as the black colour was a little bit hard to see with the dark background.

Day 12 on Project

Today, I have finished the reference list for the image I used for this website prototype. The profile pictures have all been updated to the same one in order to make look more consistent.

Reflection

I really enjoy the process of making the prototype and the biggest challenge for me in this project is to turn ideas into an actual prototype. I would love to use website programming languages to create my prototype but the fact that I am not very familiar with using these programming languages, it could make it almost impossible to achieve what I wanted to achieve. Using the Adobe XD is easier but the software itself still has still has many constraints while I was using it. Overall, it was a very valuable and interesting learning experience.
